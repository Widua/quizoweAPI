Co to za miejsce?

Jest to prowizoryczna strona startowa tylko dla ciebie, developera, który postanowił wykorzystać to API do swojego projektu, znajdziesz tutaj kompleksowe informacje w tym pełną dokumentację projektu, podstronę do testowania niektórych funkcji API(Tworzenie użytkowników, autoryzacja i tworzenie tokenu logowania, tworzenie pytania do quizu oraz losowanie pytania ) oraz informacje o pomyśle na projekt i technologiach w nim wykorzystanych.

Jak się tu poruszać?

U samej góry strony zawsze masz menu, w którym znajdziesz 5 pozycji
Menu główne jest to miejsce, w którym obecnie się znajdujesz
Dokumentacja API zawiera całkowitą dokumentację dotyczącą tego, jak z API korzystać, jakie są endpointy dla każdego możliwego działania.
Przetestuj API jest to podstrona, która umożliwia ci bez skonfigurowanego narzędzia np. postmana sprawdzić najważniejsze funkcje API, czyli rejestracja użytkowników, tworzenie pytań do quizu, oraz losowanie pytań.
O projekcie zawiera informacje na temat pomysłu na projekt, dalszych planów jego rozwoju, oraz wykorzystanych technologii webowych i nie tylko.
